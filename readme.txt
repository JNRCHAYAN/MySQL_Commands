# MySQL Commands

| No  | Work Name                   | SQL Query                                                                                                                                                                                                                      |
| --- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 1   | Clear Command line           | `system cls;`                                                                                                                                                                                                                   |
| 2   | Show All Database            | `show databases;`                                                                                                                                                                                                               |
| 3   | Create a New Database        | `create schema \`school\`;`<br>`create database CHAYAN;`                                                                                                                                                                        |
| 4   | Use specific database        | `use CHAYAN;`                                                                                                                                                                                                                   |
| 5   | Show selected database       | `select database();`                                                                                                                                                                                                            |
| 6   | Delete your database         | `drop database CHAYAN;`                                                                                                                                                                                                         |
| 7   | Create table                 | `create table user(id int, Name varchar(100));`                                                                                                                                                                                 |
| 8   | Show table details           | `desc user;`                                                                                                                                                                                                                    |
| 9   | Table data insert            | **My Sequence**: `insert into  user (Name, id) values ("chayan", 10);`<br>**Table Sequence**: `insert into  user values (103,"roy");`<br>**Multiple data**: `insert into  user values (104,"toy"),(43,"ee");`                |
| 10  | All Table data show          | `select *from user`                                                                                                                                                                                                             |
| 11  | Specific column show         | **1 column show**: `select id from user;`<br>**2 column show**: `select id, name from user`                                                                                                                                     |
| 12  | Condition                    | `select *from user where id =10;`<br>`select *from employees where emp_id = 101 || emp_id = 103;`                                                                                                                               |
| 13  | Update value                 | `update user set Name = "Korim" where id =10;`                                                                                                                                                                                  |
| 14  | Delete data from table       | **Condition**: `delete from user  where id = 100;`<br>**Full Table**: `drop table login;`                                                                                                                                       |
| 15  | Show All Table               | `show tables;`                                                                                                                                                                                                                  |
| 16  | NOT NULL                     | `create table customer (id int NOT NULL, Name varchar(100) NOT NULL);`                                                                                                                                                          |
| 17  | Default Value                | `create table employee( name varchar(100), acc_type varchar(100) default 'Svaings');`                                                                                                                                           |
| 18  | Primary Key                  | `create table employee1( acc_no int primary key, acc_type varchar(100) default 'Svaings');`                                                                                                                                     |
| 19  | Auto Increment               | `create table employee2( acc_no int primary key auto_increment, acc_type varchar(100) default 'Svaings');`                                                                                                                      |
| 20  | Table title name Change      | `Select acc_no as Account_No, acc_type as Account_type from employee2`                                                                                                                                                          |
